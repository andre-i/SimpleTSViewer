SimpleTSViewer

Simple, very simple! viewer for thingspeak channels.

    Properties:

  • Support only open channels.
  • Support show many channels, however show at one time only one channel.
  • Tune fields from channel for show( can be set field name, it measure unit, user decides what to show)
  • May set autoupdate mode
First time the "add channel" confirm appears automatically. You can add next channel use "Add channel" menu item. For it go menu -> Channels after go menu -> Add channel Inside "Add channel" confirm must set channelID. It may see about top Thingspeak channel page. It is some number.
Note : Viewer work only open channel. Private channel cannot appear in this viewer.
Next step - fill the channel properties. On top screen be channel name and autoupdate frequency. Nearby it set "done" button(Yellow). For "Name" and "Request frequency" must end witch caret return. Below is properties fields of channel. They contains:
  • field name - label for field
  • measure unit - label for field value
  • checkbox - whether show field(Default value checkbox is checked)
 All properties cannot mandatory, but how to understand what I see? After fill the properties in field must press on "ok" button. Otherwise Your values cannot saved. Filled fields to be hide. On end process press to done button(Yellow on top screen)
Press on button leave to next confirm - save channel properties. Read gotten values and if it is Ok - press on "ok" button. Otherwise press to "Cancel" and you return back for change values. In order to see channel go to menu -> Channelsb and press on "Show" button.
